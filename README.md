# LLM 多智能体社会仿真平台调研

## 2024 年相关平台与论文

| 年份   | 论文与平台                                                                                                                             | 平台实现的任务                                         | 模拟到的现象                                | 传播路径                                                                            |
| ---- | --------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------- | ------------------------------------- | ------------------------------------------------------------------------------- |
| 2024 | **LAIDSim 与 LAID：An LLM enhanced Agent based Simulation Tool for Information Propagation** <br> [论文链接][p2024-1]                   | 构建 LLM 增强的影响力扩散模型，并提供 Web 可视化工具，用于模拟社交网络中的信息传播。 | 信息扩散、影响力覆盖、用户态度变化、语言级交互。              | 初始信息源进入社交网络。LLM 用户代理接收信息后，判断是否接受、转发或改变观点。信息继续通过社交关系扩散，并在可视化界面中展示传播范围和态度变化。      |
| 2024 | **Y Social: an LLM powered Social Media Digital Twin** <br> [论文链接][p2024-2]                                                       | 构建社交媒体数字孪生平台，模拟用户互动、内容生成、内容传播和网络动态。             | 用户参与、信息扩散、平台政策影响、在线环境中的涌现现象。          | 真实或模拟新闻进入平台。LLM 用户生成回应，并产生点赞、评论和转发等互动。网络结构和推荐机制影响内容曝光，最终形成内容扩散和群体反应。            |
| 2024 | **OASIS: Open Agent Social Interaction Simulations with One Million Agents** <br> [论文链接][p2024-3]                                 | 构建可扩展社交媒体仿真器，模拟 X 和 Reddit 等平台，支持百万级用户仿真。       | 信息传播、群体极化、从众效应、不同规模下的群体动力学。           | 帖子进入动态社交环境。推荐系统根据兴趣和热度进行分发。代理执行关注、评论和转发等行为。社交网络持续更新，信息继续传播，并可能引发极化或从众效应。        |
| 2024 | **Language Evolution for Evading Social Media Regulation via LLM based Multi agent Simulation** <br> [论文链接][p2024-4]              | 模拟用户在严格内容监管下如何演化出隐语和编码语言来规避监管。                  | 语言演化、规避审查、信息准确性变化、监管对抗。               | 参与者代理尝试表达信息。监管代理进行检测和拦截。参与者根据失败经验调整表达方式，逐渐形成隐语、替代表达和语义迁移，使信息继续传播。               |
| 2024 | **Large Language Model driven Multi Agent Simulation for News Diffusion Under Different Network Structures** <br> [论文链接][p2024-5] | 模拟不同网络结构下新闻和假新闻传播，并测试干预策略。                      | 假新闻扩散、人格特质影响、网络结构影响、关键节点阻断效果、真实性提示效果。 | 新闻由种子节点发布。代理根据人格特质和心理倾向判断是否转发。信息沿不同网络结构扩散，形成局部传播或全局传播。干预策略通过阻断关键节点或提供真实性提示进行纠偏。 |

## 2025 年相关平台与论文

| 年份   | 论文与平台                                                                                                                   | 平台实现的任务                               | 模拟到的现象                                  | 传播路径                                                                           |
| ---- | ----------------------------------------------------------------------------------------------------------------------- | ------------------------------------- | --------------------------------------- | ------------------------------------------------------------------------------ |
| 2025 | **AgentSociety: Large Scale Simulation of LLM Driven Generative Agents** <br> [论文链接][p2025-1]                           | 构建大规模社会仿真器，模拟一万多个智能体的日常生活、社会互动和环境互动。  | 极化、煽动性信息传播、基本收入政策影响、飓风冲击、城市可持续性。        | 城市和社会环境提供外部约束。代理执行日常活动并进行社交互动。信息在关系网络和环境事件中传播，推动群体态度变化，最终形成社会级现象。              |
| 2025 | **Simulating Rumor Spreading in Social Networks using LLM Agents** <br> [论文链接][p2025-2]                                 | 构建 LLM 代理谣言传播框架，在不同网络结构中模拟谣言扩散。       | 谣言传播、网络结构差异、人格差异、传播策略差异，最高可影响百分之八十三的代理。 | 谣言由种子节点发布。邻居代理看到信息后，结合 persona、历史帖子和自身信念判断接受、拒绝或继续传播。新帖子进入邻居信息流，经过多轮迭代后形成扩散结果。 |
| 2025 | **MOSAIC: Modeling Social AI for Content Dissemination and Regulation in Multi Agent Simulations** <br> [论文链接][p2025-3] | 构建开源社交网络仿真框架，让生成式语言代理预测用户点赞、分享和举报等行为。 | 内容扩散、虚假信息传播、内容治理、事实核查策略效果、群体参与度变化。      | 内容首先被发布到社交网络。代理基于记忆、自我反思和推理决定点赞、分享或举报。内容沿社交图传播，事实核查或社区治理策略介入后，扩散速度和参与度发生变化。    |
| 2025 | **Decoding Echo Chambers: LLM Powered Simulations Revealing Polarization in Social Networks** <br> [论文链接][p2025-4]      | 构建 LLM 社会意见网络，用于评估和缓解极化。              | 回声室、意见极化、同质化互动、主动提示和被动提示的缓解效果。          | 用户处于特定网络结构中。推荐算法决定用户可见内容和交互对象。代理基于推理更新观点。相似观点逐渐聚集，最终强化回声室和意见极化。                |
| 2025 | **LLM AIDSim: LLM Enhanced Agent Based Influence Diffusion Simulation in Social Networks** <br> [论文链接][p2025-5]         | 在 LAIDSim 基础上进一步模拟影响力扩散、话题演化和用户话语变化。  | 话题聚集、集体讨论焦点收敛、城市间扩散差异。                  | 公共议题或新闻进入网络。用户代理生成语言级回应。影响力扩散模型推动邻居参与讨论。话题相似度逐步变化，集体讨论逐渐集中到少数主导话题。             |
| 2025 | **Understanding Dynamic Diffusion Process of LLM based Agents under Information Asymmetry** <br> [论文链接][p2025-6]        | 研究信息不对称开放环境下 LLM 智能体的信息扩散过程。          | 信息茧房、信息差演化、社会资本积累、选择性注意。                | 外部信息刺激首先进入小规模代理群体。代理产生回应并扩大互动范围。群体规模增加后形成信息圈。关系变化影响可见信息，最终出现信息茧房和信息差。          |

| 2026 | **PolicySim: An LLM Based Agent Social Simulation Sandbox for Proactive Policy Optimization** <br> [论文链接][p2026-1]                           | 构建社交平台政策沙盒，用于提前评估推荐、过滤和干预政策对平台生态的影响。 | 推荐系统导致的回声室、极化、平台干预副作用、政策优化效果。     | 用户行为首先进入平台环境。平台推荐、过滤和干预策略影响用户可见内容。用户再次产生反馈、互动和观点变化。平台根据反馈继续调整策略，形成用户行为与平台干预之间的双向动态循环。 |
| 2026 | **WhatIf: Interactive Exploration of LLM Powered Social Simulations for Policy Reasoning** <br> [论文链接][p2026-2]                              | 构建交互式政策推演系统，支持政策制定者实时控制、检查和比较社会仿真结果。 | 灾害疏散中的协调行为、信息理解、采纳行为、规划漏洞暴露。      | 政策或应急信息被发布到模拟环境中。人群代理解释信息，并进行个体协调和行动采纳。系统实时展示个体结果和群体结果。政策制定者根据反馈调整方案，并比较不同分支结果。       |
| 2026 | **LLM Agent based Social Simulation for Attitude Diffusion and discourse simulator** <br> [论文链接][p2026-3]                                    | 构建开源框架，模拟重大事件后公众态度如何变化。              | 态度扩散、移民议题极化、信念演化、事件驱动的观点变化。       | 真实新闻和事件时间线进入模拟系统。LLM 生成社交媒体帖子，并解释其中的观点。代理根据接收到的信息更新信念。信息在小世界网络中传播，最终形成态度扩散和极化。        |
| 2026 | **Public opinion dissemination simulation based on large language model multi agent systems** <br> [论文链接][p2026-4]                           | 构建 LLM 多智能体舆情传播系统，模拟公共政策和食品安全等场景。    | 舆情潜伏、舆情爆发、舆情衰退、行为异质性、话题倾向变化、角色差异。 | 系统首先完成场景初始化。不同角色代理生成内容。全局信息共享池存储舆情信息。代理基于记忆和角色执行点赞、评论、转发和发帖等行为。宏观反馈继续影响下一轮传播。         |
| 2026 | **LLM Powered Social Digital Twins: A Framework for Simulating Population Behavioral Response to Policy Interventions** <br> [论文链接][p2026-5] | 构建社会数字孪生框架，预测人群对政策干预的行为反应。           | 疫情政策响应、群体行为概率变化、政策反事实推演。          | 政策信号被输入系统。具有人口统计属性和心理属性的代理接收政策信号。代理输出多维行为概率。校准层将代理结果映射到真实人口指标，并用于反事实政策比较。             |
