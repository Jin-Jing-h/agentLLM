# LLM 多智能体社会仿真平台调研

## 2024 年相关平台与论文

| 年份   | 论文与平台                                                                                                                             | 平台实现的任务                                                          | 模拟到的现象                                                   | 传播路径                                                                                                      |
| ---- | --------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------- | -------------------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| 2024 | **LAIDSim 与 LAID：An LLM enhanced Agent based Simulation Tool for Information Propagation** <br> [论文链接][p2024-1]                   | 做了一个可以观察信息传播过程的仿真工具。研究者可以把一条信息放进社交网络中，观察不同用户代理是否会接收、相信或继续传播这条信息。 | 可以看到信息在社交网络中怎样一步步扩散，也可以看到哪些用户影响力更大，以及用户态度是否发生变化。         | 一条信息先从少数用户开始传播。每个用户代理读到信息后，会根据自己的判断决定是否接受、是否转发、是否改变看法。随后信息继续传给更多相连的用户，系统会展示传播范围和态度变化。                     |
| 2024 | **Y Social: an LLM powered Social Media Digital Twin** <br> [论文链接][p2024-2]                                                       | 做了一个类似真实社交媒体的数字平台。平台里的用户由大模型代理扮演，可以发帖、回复、互动，并形成社交网络。             | 可以观察用户参与、内容传播、平台规则影响，以及社交媒体中可能出现的群体反应。                   | 新闻或帖子先进入平台。用户代理看到内容后生成回复，并产生点赞、评论和转发等行为。平台中的关系网络和推荐机制会影响谁能看到内容，最后形成更大范围的讨论和传播。                            |
| 2024 | **OASIS: Open Agent Social Interaction Simulations with One Million Agents** <br> [论文链接][p2024-3]                                 | 做了一个大规模社交媒体仿真器，可以模拟 X 和 Reddit 等平台上的大量用户行为，最多支持百万级用户。            | 可以观察信息传播、群体极化、从众效应，以及不同用户规模下群体行为的变化。                     | 帖子先进入社交平台环境。推荐系统把帖子推送给感兴趣或可能参与的用户。用户代理可以关注、评论和转发。随着互动增加，社交关系和内容热度不断变化，信息可能被更多人看到，也可能引发极化或从众现象。            |
| 2024 | **Language Evolution for Evading Social Media Regulation via LLM based Multi agent Simulation** <br> [论文链接][p2024-4]              | 模拟用户在内容监管较严格的环境下，如何逐渐改变表达方式来继续交流。                                | 可以观察隐语、替代表达和编码语言如何出现，也可以看到用户表达方式和监管之间的对抗过程。              | 用户代理先尝试表达某些信息。监管代理会检查内容是否违规。如果内容被拦截，用户代理会调整说法，例如换词、绕开敏感表达或使用隐语。经过多轮交流后，用户可能形成新的表达方式，让信息继续传递。              |
| 2024 | **Large Language Model driven Multi Agent Simulation for News Diffusion Under Different Network Structures** <br> [论文链接][p2024-5] | 模拟新闻和假新闻在不同社交网络结构中的传播过程，并测试不同干预方法是否有效。                           | 可以观察假新闻如何扩散，用户性格和网络结构如何影响传播，以及阻断关键用户和提示新闻真实性是否能减少错误信息传播。 | 新闻先由初始用户发布。其他用户代理看到后，会根据自己的性格、兴趣和判断决定是否讨论或转发。信息沿着不同网络结构传播，可能只在局部扩散，也可能扩散到整个网络。干预方法会通过阻断重要节点或提醒新闻真实性来减少传播。 |

## 2025 年相关平台与论文

| 年份   | 论文与平台                                                                                                                   | 平台实现的任务                                      | 模拟到的现象                                                 | 传播路径                                                                                            |
| ---- | ----------------------------------------------------------------------------------------------------------------------- | -------------------------------------------- | ------------------------------------------------------ | ----------------------------------------------------------------------------------------------- |
| 2025 | **AgentSociety: Large Scale Simulation of LLM Driven Generative Agents** <br> [论文链接][p2025-1]                           | 做了一个大规模社会仿真平台，让一万多个智能体在城市环境中生活、行动、交流和受到环境影响。 | 可以观察社会极化、煽动性信息传播、基本收入政策影响、飓风等外部冲击，以及城市可持续性相关现象。        | 智能体生活在一个模拟城市中，会进行日常活动和社交互动。信息会通过人与人之间的关系和外部事件传播。随着互动不断发生，个体态度和行为逐渐变化，最终形成群体层面的社会现象。             |
| 2025 | **Simulating Rumor Spreading in Social Networks using LLM Agents** <br> [论文链接][p2025-2]                                 | 做了一个谣言传播仿真框架，用大模型代理模拟谣言在不同社交网络中的扩散。          | 可以观察谣言是否会扩散、会影响多少人，以及网络结构、用户画像和传播策略对谣言传播的影响。           | 谣言先由少数用户发布。邻居用户看到后，会根据自己的身份设定、过去看到的内容和已有信念决定相信、不相信或继续传播。新的传播内容会进入更多用户的信息流，经过多轮传播后形成最终扩散范围。      |
| 2025 | **MOSAIC: Modeling Social AI for Content Dissemination and Regulation in Multi Agent Simulations** <br> [论文链接][p2025-3] | 做了一个开源社交网络仿真框架，用用户代理预测人们面对内容时会点赞、分享还是举报。     | 可以观察内容如何传播，虚假信息如何扩散，不同内容治理策略是否能降低不真实内容的影响，以及用户参与度如何变化。 | 内容先发布到社交网络中。用户代理看到内容后，会结合自身画像和推理过程决定点赞、分享或举报。内容沿社交关系继续传播。事实核查或社区治理策略介入后，传播速度和用户参与会发生变化。         |
| 2025 | **Decoding Echo Chambers: LLM Powered Simulations Revealing Polarization in Social Networks** <br> [论文链接][p2025-4]      | 做了一个社会意见网络仿真系统，用来研究用户观点如何变化，以及如何缓解极化。        | 可以观察回声室、意见极化、相似观点聚集，以及主动提示和被动提示对缓解极化的作用。               | 用户处在不同类型的社交网络中。推荐算法会决定用户看到哪些内容、接触哪些人。用户代理根据看到的信息进行推理并更新观点。相似观点的用户逐渐聚集，回声室和意见极化可能被加强，也可能被提示策略缓解。 |
| 2025 | **LLM AIDSim: LLM Enhanced Agent Based Influence Diffusion Simulation in Social Networks** <br> [论文链接][p2025-5]         | 在影响力扩散模型中加入大模型代理，用来模拟人们如何围绕公共话题产生回应并影响他人。    | 可以观察话题如何聚集，集体讨论如何逐渐集中到少数主题，以及不同城市或群体之间的传播差异。           | 公共议题或新闻先进入社交网络。用户代理会用自然语言表达看法。影响力扩散模型决定哪些邻居会继续参与讨论。随着交流增加，大家关注的话题逐渐靠近，讨论可能集中到少数主要话题上。           |
| 2025 | **Understanding Dynamic Diffusion Process of LLM based Agents under Information Asymmetry** <br> [论文链接][p2025-6]        | 研究在信息不对称环境中，不同智能体怎样接触、选择和传播信息。               | 可以观察信息茧房、信息差、社会资本积累，以及用户注意力选择带来的差异。                    | 外部信息先进入一个小规模群体。智能体会选择关注哪些信息，并与其他人交流。随着群体扩大，智能体之间的关系不断变化，不同人能看到的信息也不同。最终可能形成信息圈层、信息差和信息茧房。       |

## 2026 年相关平台与论文

| 年份   | 论文与平台                                                                                                                                        | 平台实现的任务                                        | 模拟到的现象                                           | 传播路径                                                                                  |
| ---- | -------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------- | ------------------------------------------------ | ------------------------------------------------------------------------------------- |
| 2026 | **PolicySim: An LLM Based Agent Social Simulation Sandbox for Proactive Policy Optimization** <br> [论文链接][p2026-1]                           | 做了一个社交平台政策沙盒，用来在政策真正上线前，提前测试推荐、过滤和干预策略会产生什么影响。 | 可以观察推荐系统是否会加重回声室和极化，也可以观察平台干预是否有效，是否会带来副作用。      | 用户先在平台中浏览和互动。平台策略会影响用户看到什么内容。用户看到内容后继续反馈、互动并改变观点。平台再根据用户反馈调整干预策略，形成用户行为和平台政策相互影响的循环。  |
| 2026 | **WhatIf: Interactive Exploration of LLM Powered Social Simulations for Policy Reasoning** <br> [论文链接][p2026-2]                              | 做了一个交互式政策推演系统，让政策制定者可以实时调整方案、观察结果并比较不同方案。      | 可以观察灾害疏散中人群是否理解信息、是否协调行动、是否采纳建议，以及原有计划中可能被忽略的问题。 | 政策或应急通知先发布给人群代理。人群代理会理解信息，并决定是否行动、如何协调。系统实时展示个人和群体结果。政策制定者可以根据结果修改方案，再比较不同方案的效果。      |
| 2026 | **LLM Agent based Social Simulation for Attitude Diffusion and discourse simulator** <br> [论文链接][p2026-3]                                    | 做了一个开源工具，用来模拟重大事件发生后，公众态度如何随着新闻和社交讨论发生变化。      | 可以观察公众态度扩散、移民议题极化、信念变化，以及现实事件如何推动观点变化。           | 真实新闻和事件时间线先进入模拟系统。大模型生成社交媒体帖子，并识别帖子中的观点。智能体看到信息后更新自己的信念。信息在小世界网络中继续传播，最后形成态度扩散和观点极化。  |
| 2026 | **Public opinion dissemination simulation based on large language model multi agent systems** <br> [论文链接][p2026-4]                           | 做了一个公共舆情传播仿真系统，用来模拟公共政策和食品安全等场景下舆情如何发展。        | 可以观察舆情从潜伏到爆发再到衰退的过程，也可以观察不同角色、不同观点和不同行为带来的传播差异。  | 系统先设定舆情场景。不同角色的智能体生成观点和内容。信息被放入共享信息池中。智能体根据自身角色和记忆进行点赞、评论、转发和发帖。上一轮传播结果会影响下一轮舆情变化。    |
| 2026 | **LLM Powered Social Digital Twins: A Framework for Simulating Population Behavioral Response to Policy Interventions** <br> [论文链接][p2026-5] | 做了一个社会数字孪生框架，用虚拟人群模拟真实人群面对政策干预时可能会怎样反应。        | 可以观察疫情政策下人群行为的变化，也可以比较不同政策方案可能产生的结果。             | 政策信号先输入系统。带有人口特征和心理特征的虚拟个体接收政策信息。每个个体输出可能采取的行为概率。系统再把个体结果汇总并校准到真实人口指标上，用来比较不同政策的可能效果。 |
